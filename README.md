# Practical Linear Algebra

Repository ini berisi reproduksi kode dan rangkuman dari buku **Practical Linear Algebra for Data Science (O’Reilly)**.

## Tujuan
Memahami konsep dasar linear algebra yang digunakan dalam machine learning melalui implementasi Python dan visualisasi.

## Struktur Repository

### Chapter 1 – Introduction
Pengantar linear algebra dan perannya dalam data science.

### Chapter 2 – Vectors
Konsep vektor dan representasi data.

### Chapter 3 – Vector Operations
Operasi vektor seperti magnitude dan dot product.

### Chapter 4 – Matrices
Representasi data dalam bentuk matriks.

### Chapter 5 – Matrix Operations
Operasi matriks, determinan, dan invers.

### Chapter 6 – QR Decomposition
Pemecahan matriks menjadi Q dan R.

### Chapter 7 – Matrix Rank
Menentukan rank dan independensi linear.

### Chapter 8 – Matrix Inverse
Konsep invers dan identity matrix.

### Chapter 9 – Orthogonality
Hubungan orthogonal antar vektor.

### Chapter 10 – Row Reduction
Penyelesaian sistem linear menggunakan RREF.

## Kesimpulan
Linear algebra merupakan dasar penting dalam machine learning, terutama dalam representasi data dan transformasi.
